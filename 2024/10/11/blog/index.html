个人博客搭建心路历程

Created2024-10-11|Updated2024-10-15
|Post Views:

1.首先到官网安装Node.js，按照提示完成安装。
2.接着到git官网安装git，选择64位，如果无法正常打开网页，可搜索watt toolkit，下载Steam++，对网络继续加速即可，后面按照提示安装。
3.安装VScode，也是按照提示安装，打开后安装中文插件，方便进行详细配置。
4.用命令提示符检查是否安装了全部必备软件。（如果卡住，输入 npm install -g cnpm –registry=https://registry.npmmirror.com ，记得在npm前加c）
5.进入VScode点击新建终端，输入cd heo 进入文件夹，输入cnpm install安装依赖，完成博客框架初始化。
6.在hexo官网搜索Butterfly，进入该项目的开源地址，复制第一条代码 git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly ，粘贴到VScode上运行，如果报错，就输入 git config –global http.sslVerify false，关闭证书认证，接着到开源项目那复制cnpm install hexo-renderer-pug hexo-renderer-stylus，安装渲染器，在themes中找到config.yml文件，在99行主题这块把landscape改成Butterfly，再Ctrl+S保存，输入hexo cl清除。
7.前面有报错，输入cnpm install hexo-util –save，再次输入hexo cl清除，接着输入hexo g生成静态文件。
8.在博客根目录heo（自己命名的），创建文件夹_config.butterfly.yml，找到_config.yml，全部复制粘贴到新创建的文件夹，保存一下。
9.接着开始部署到互联网上，到GitHub注册账号，登入后新建仓库，名称为blog。
10.回到VScode中找到_config.yml，在105行输入 repo：仓库地址（复制github上的网址），回车到106行填一下分支branch： main，保存，为了防止出错再次用第六条提到的代码禁用证书，接着依次输入hexo cl，hexo g，hexo d，会出现登入框，授权即可。
11.最后，打开Zeabur官网，授权后新建项目，选择一个服务（建议选择Germany，免费），选择仓库后部署，部署完成后，在Networking，自定义一个域名，即可通过域名访问个人博客。
